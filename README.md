# 🐱 CatPhotoApp

My first HTML project built while learning web development through the freeCodeCamp Responsive Web Design curriculum.

This project is a simple webpage that showcases cat photos, lists of things cats love and hate, and basic HTML elements such as headings, paragraphs, links, images, lists, and semantic tags.

## 📖 About the Project

CatPhotoApp was created as a hands-on exercise to practice:

- HTML document structure
- Semantic HTML elements
- Headings and paragraphs
- Hyperlinks
- Images and alternative text
- Ordered and unordered lists
- Figure and figcaption elements
- Basic webpage organization

## 🚀 Features

- Display cat photos from external sources
- Clickable links to cat-related websites
- Lists of things cats love and hate
- Image captions using semantic HTML
- Clean and beginner-friendly structure

## 🛠️ Technologies Used

- HTML5

## 🎯 Learning Outcomes

Through this project, I learned:

- How to create a valid HTML5 document
- The purpose of semantic elements like `<main>`, `<section>`, `<figure>`, and `<footer>`
- How to add images and hyperlinks
- How to organize content using lists
- The importance of accessibility through `alt` text
